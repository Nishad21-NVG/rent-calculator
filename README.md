# Rent Calculator

A simple Python program to calculate shared rent, food, and electricity expenses among roommates.

## Features

- Calculates each person's share based on input rent, food, electricity usage, and number of persons.
- Takes user input for all expenses and number of persons.
- Provides clear output showing the breakdown and each personâ€™s amount due.

## Usage

1. Run the `Rentcal.py` script in your terminal or command line.
2. Enter the required details: rent, food cost, electricity units, charge per unit, and number of roommates.
3. View the calculation results for each person.
